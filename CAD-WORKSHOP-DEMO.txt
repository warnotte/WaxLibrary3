Option 1: Depuis votre IDE (RECOMMANDÉ)
   1. Ouvrez IntelliJ IDEA / Eclipse / NetBeans
   2. Naviguez vers:
      io.github.warnotte.waxlib3.W2D.PanelGraphique.demo.cadworkshop.CADWorkshopDemo
   3. Clic-droit → Run As → Java Application

Option 2: Script Windows
   Double-cliquez sur: run-cadworkshop-demo.bat

Option 3: Ligne de commande
   mvn exec:java -Dexec.mainClass="io.github.warnotte.waxlib3.W2D.PanelGraphique.demo.cadworkshop.CADWorkshopDemo"

   🖱️ Bouton milieu     - Déplacer la vue (pan)
   🖱️ Bouton gauche     - Sélectionner
   🖱️ Drag              - Rectangle de sélection

Clavier
   [A]                  - Animation ON/OFF
   [T]                  - Changer mode de test
   [M]                  - Mode mesure
   [S][P][R][L][K]      - Changer d'outil
   [SHIFT]              - Ajouter à la sélection
   [CTRL]               - Retirer de la sélection
   [Flèches]            - Déplacer la vue
   [+/-]                - Zoom
